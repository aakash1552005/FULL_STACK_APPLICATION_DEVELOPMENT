\## 6. Pay-As-You-Use Pricing Model



\### AWS Pricing

\- Pay only for what you use

\- No minimum fees or upfront costs

\- Example: EC2 charged per second/hour

\- Free tier available for 12 months



\### Google Cloud Pricing

\- Per-second billing

\- Sustained use discounts automatically applied

\- Committed use discounts up to 57%

\- $300 free credits for new users



\### Azure Pricing

\- Pay-as-you-go model

\- Reserved instances for discounts up to 72%

\- Hybrid benefit for existing licenses

\- Free tier with 12 months of popular services

\---



\## 8. Conclusion



All three providers offer similar core services with different strengths:

\- \*\*AWS\*\* - Most mature, largest ecosystem

\- \*\*Google Cloud\*\* - Best for AI/ML and data analytics  

\- \*\*Azure\*\* - Best for enterprises using Microsoft products



The pay-as-you-use model means you only pay for resources consumed,

making cloud computing cost-effective for businesses of all sizes.
