\# Task 22 - Cloud Service Providers Comparison



\## 1. Major Cloud Service Providers

\- \*\*AWS\*\* (Amazon Web Services)

\- \*\*Google Cloud Platform (GCP)\*\*

\- \*\*Microsoft Azure\*\*



\---



\## 2. Compute Services



| Service | AWS | Google Cloud | Azure |

|---------|-----|--------------|-------|

| Virtual Machines | EC2 | Compute Engine | Virtual Machines |

| Serverless | Lambda | Cloud Functions | Azure Functions |

| Containers | ECS/EKS | GKE | AKS |



\---



\## 3. Storage Services



| Service | AWS | Google Cloud | Azure |

|---------|-----|--------------|-------|

| Object Storage | S3 | Cloud Storage | Blob Storage |

| Block Storage | EBS | Persistent Disk | Managed Disks |

| File Storage | EFS | Filestore | Azure Files |



\---



\## 4. Database Services



| Service | AWS | Google Cloud | Azure |

|---------|-----|--------------|-------|

| Relational DB | RDS | Cloud SQL | Azure SQL |

| NoSQL | DynamoDB | Firestore | Cosmos DB |

| In-Memory Cache | ElastiCache | Memorystore | Azure Cache |



\---



\## 5. Networking Services



| Service | AWS | Google Cloud | Azure |

|---------|-----|--------------|-------|

| Virtual Network | VPC | VPC | Virtual Network |

| Load Balancer | ELB | Cloud Load Balancing | Azure Load Balancer |

| CDN | CloudFront | Cloud CDN | Azure CDN |

| DNS | Route 53 | Cloud DNS | Azure DNS |



\---



\## 6. Pay-As-You-Use Pricing Model



| Provider | Pricing Model | Free Tier | Discount Options |

|----------|--------------|-----------|-----------------|

| AWS | Per second/hour | 12 months | Reserved instances up to 75% |

| Google Cloud | Per second | $300 credits | Sustained use discounts up to 57% |

| Azure | Per minute | 12 months | Reserved instances up to 72% |



\---



\## 7. Comparison Summary



| Feature | AWS | Google Cloud | Azure |

|---------|-----|--------------|-------|

| Market Share | 32% | 11% | 22% |

| Global Regions | 33 | 40 | 60+ |

| Best For | Enterprise | AI/ML/Data | Microsoft ecosystem |



\---



\## 8. Conclusion

\- \*\*AWS\*\* - Most mature, largest ecosystem

\- \*\*Google Cloud\*\* - Best for AI/ML and data analytics

\- \*\*Azure\*\* - Best for enterprises using Microsoft products
